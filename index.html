PL · Lineal
Método Simplex — Gran M
max cᵀx s.a. Ax ≤ b, x ≥ 0
Algoritmo iterativo que recorre los vértices del poliedro factible para maximizar o minimizar una función objetivo lineal. La variante Gran M penaliza variables artificiales con un costo M → ∞, evitando soluciones infactibles.

Redes · Flujo
Flujo Máximo
max Σ fᵢⱼ s.a. 0 ≤ fᵢⱼ ≤ cᵢⱼ
Determina la cantidad máxima de flujo que puede circular desde la fuente (s) hasta el sumidero (t) en una red de capacidades. Se basa en el teorema de corte mínimo–flujo máximo (Ford-Fulkerson / Edmonds-Karp).

Método Gráfico

Solución visual de problemas de programación lineal con dos variables de decisión. Traza las restricciones, identifica la región factible y determina el vértice donde la función objetivo alcanza su óptimo.

Asignación
Método Húngaro
min Σᵢⱼ cᵢⱼ xᵢⱼ s.a. asignación perfecta
Resuelve el problema de asignación óptima en tiempo polinomial O(n³). Reduce la matriz de costos por filas y columnas, y encuentra una asignación perfecta mediante cobertura de ceros. Ideal para asignación de recursos, trabajos o rutas.

PNL · Óptimo
Condiciones KKT
∇f = Σ λᵢ∇gᵢ + Σ μⱼ∇hⱼ, λᵢgᵢ=0
Condiciones de Karush-Kuhn-Tucker: criterio necesario (y suficiente bajo convexidad) de optimalidad para programación no lineal con restricciones de igualdad y desigualdad. Generalización de los multiplicadores de Lagrange.

Programación No Convexa
f no convexa → múltiples mínimos locales
Optimización de funciones con regiones factibles o funciones objetivo no convexas, donde pueden existir múltiples óptimos locales. Se aplican métodos de búsqueda global, penalización o relajación para encontrar el óptimo global.

PNL · Raíces
Método de Newton-Raphson
xₙ₊₁ = xₙ − f(xₙ) / f′(xₙ)
Método iterativo de convergencia cuadrática para encontrar raíces de funciones no lineales o resolver sistemas de ecuaciones. Parte de una aproximación inicial x₀ y refina la solución usando la tangente de la función en cada iteración.

Método de Transporte

PNL · Convexa
Programación Convexa
f convexa, gᵢ convexa → mín. local = global
Clase de PNL donde la función objetivo y la región factible son convexas, garantizando que todo mínimo local sea global. Incluye programación cuadrática convexa, SOCP y SDP. Se resuelve eficientemente con métodos de punto interior.

PNL · Cuadrática
Programación Cuadrática
min ½xᵀQx + cᵀx s.a. Ax ≤ b
Optimiza una función objetivo cuadrática sujeta a restricciones lineales. Si Q es semidefinida positiva el problema es convexo. Se resuelve con el algoritmo de Wolf, gradiente reducido o métodos de punto activo.

PNL · Separable
Programación Separable
min Σᵢ fᵢ(xᵢ) s.a. Σᵢ gᵢⱼ(xᵢ) ≤ bⱼ
Caso especial de PNL donde la función objetivo y las restricciones son sumas de funciones univariadas. Se linealiza mediante aproximación por tramos (PLSep) y se resuelve como un problema de PL ampliado, siendo muy eficiente en la práctica.

Ruta Más Corta
d(v) = min{d(u) + w(u,v)} ∀ v ∈ V
Encuentra el camino de menor costo entre un origen y uno o todos los destinos en una red ponderada. Algoritmos: Dijkstra (pesos positivos O((V+E)logV)), Bellman-Ford (pesos negativos O(VE)) o Floyd-Warshall (todos los pares O(V³)).
