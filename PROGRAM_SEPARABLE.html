¿Qué es la Programación Separable?

Un problema es separable cuando f(x) se puede escribir como suma de funciones de una sola variable:

Z = f₁(x₁) + f₂(x₂) + … + fₙ(xₙ)

Esto permite linearizar cada fⱼ de forma independiente y resolver con Programación Lineal.

Forma estándar

Maximizar:

Z = Σⱼ fⱼ(xⱼ)

Sujeto a:

Σⱼ aᵢⱼ xⱼ ≤ bᵢ ∀i
xⱼ ≥ 0

Linearización PWL

Para cada fⱼ(xⱼ) se definen k+1 puntos de quiebre y ponderaciones λ:

xⱼ = Σₖ aⱼₖ·λⱼₖ
fⱼ(xⱼ) ≈ Σₖ fⱼ(aⱼₖ)·λⱼₖ

Con: λⱼₖ ≥ 0, Σₖ λⱼₖ = 1 y condición SOS2.

Condición SOS2

A lo más dos variables λ adyacentes pueden ser positivas simultáneamente, garantizando que la aproximación corresponde a un único segmento activo.

Para funciones cóncavas (maximización), SOS2 se cumple automáticamente en Simplex.

Ejemplo — Ejercicio 4 del curso

1

Separabilidad: Z = (20x₁ − x₁²) + (16x₂ − 2x₂²). Cada término depende de una sola variable ✓

2

Puntos de quiebre para x₁, x₂ ∈ [0,8]: se eligen 0, 2, 4, 6, 8 (k=4 segmentos)

3

Evaluar: f₁(0)=0, f₁(2)=36, f₁(4)=64, f₁(6)=84, f₁(8)=96 · f₂(0)=0, f₂(2)=24, f₂(4)=32, f₂(6)=24, f₂(8)=0

4

PL equivalente: maximizar Σ fⱼ(aⱼₖ)·λⱼₖ, sujeto a Σ aⱼₖ·λⱼₖ satisface x₁+x₂≤8, Σλ=1, SOS2

5

Óptimo: x₁*=10, x₂*=4 → Z*=164 (el solucionador lo obtiene automáticamente)
