    ¿Qué son las condiciones KKT?

    Las condiciones de Karush–Kuhn–Tucker (KKT) son condiciones de primer orden necesarias para que una solución de un problema de optimización con restricciones sea óptima. Son una generalización de los multiplicadores de Lagrange para el caso de restricciones de desigualdad.

    Fueron desarrolladas por William Karush en 1939 y de forma independiente por Harold Kuhn y Albert Tucker en 1951.

    Problema General

    Minimizar (o maximizar) f(x)
    sujeto a: gᵢ(x) ≤ 0, i = 1, ..., m (restricciones de desigualdad)
    hⱼ(x) = 0, j = 1, ..., p (restricciones de igualdad)

    Lagrangiano

    Se construye la función Lagrangiana combinando la función objetivo con las restricciones mediante multiplicadores:

    L(x, μ, λ) = f(x) + Σ μᵢ·gᵢ(x) + Σ λⱼ·hⱼ(x)

    Donde μᵢ son los multiplicadores de KKT (para desigualdades) y λⱼ son los multiplicadores de Lagrange (para igualdades).

    Las 4 Condiciones KKT

    #CondiciónExpresión MatemáticaNombre
    1 Estacionariedad ∇f(x*) + Σ μᵢ·∇gᵢ(x*) + Σ λⱼ·∇hⱼ(x*) = 0 Gradiente del Lagrangiano = 0
    2 Factibilidad Primal gᵢ(x*) ≤ 0, hⱼ(x*) = 0 Se satisfacen todas las restricciones
    3 Factibilidad Dual μᵢ ≥ 0 para todo i Multiplicadores no negativos
    4 Holgura Complementaria μᵢ · gᵢ(x*) = 0 para todo i Restricción activa ↔ multiplicador > 0

    Condiciones de Suficiencia

    Las condiciones KKT son necesarias bajo ciertas calificaciones de restricciones (LICQ, MFCQ, etc.). Son suficientes cuando:

    • • f(x) es convexa → problema de minimización convexo
    • • f(x) es cóncava → problema de maximización
    • • gᵢ(x) son convexas y hⱼ(x) son afines

    Interpretación Económica de μᵢ

    Los multiplicadores de KKT tienen una interpretación importante: μᵢ representa el costo marginal o precio sombra de la restricción i-ésima. Indica cuánto mejoraría la función objetivo si se relaja la restricción i en una unidad.

    μᵢ = -∂f*(b)/∂bᵢ (tasa de cambio del óptimo respecto a la holgura)

    📐 Clásico KKT básico
    Min x1² + x2²
    s.a. x1 + x2 ≥ 4
    x1, x2 ≥ 0
    💼 Maximización cuadrática
    Max 4x1 + 5x2 - x1² - x2²
    s.a. 2x1 + x2 ≤ 8
    x1, x2 ≥ 0
    🏭 Producción óptima
    Min x1² + 2x2² - 6x1
    s.a. x1 + x2 = 3
    📊 Dos desigualdades
    Min (x1-3)² + (x2-2)²
    s.a. x1 + x2 ≤ 4
    x1 ≤ 2
    🔬 No lineal clásico
    Max x1·x2
    s.a. x1 + x2 ≤ 6
    x1, x2 ≥ 0
    ⚡ Minimizar distancia
    Min x1² + x2²
    s.a. x1 + 2x2 = 6
    Cómo Formular el Problema
    1 FUNCIÓN OBJETIVO
    Escribe la función que deseas minimizar o maximizar usando x1, x2, x3. Puedes usar potencias (x1^2), multiplicación (x1*x2), y funciones como sqrt(), log(), exp().
    Ej: x1^2 + x2^2 - 4*x1
    2 RESTRICCIONES
    Cada restricción se ingresa en tres partes: el lado izquierdo g(x), el signo (≤, ≥ o =), y el lado derecho (cualquier número). Ya no tienes que mover el RHS manualmente al otro lado.
    Ej: x1 + x2 ≤ 6 → escribe "x1 + x2" | ≤ | "6"
    Ej: 2*x1 - x2 ≥ -3 → escribe "2*x1 - x2" | ≥ | "-3"
    3 NO NEGATIVIDAD
    Si necesitas x1 ≥ 0, agrégala directamente: escribe x1 | ≥ | 0. Sencillo.
